# Гипотеза 3 — Влияние трансформаций на качество прогноза временных рядов

## Гипотеза

> Для разных типов временных рядов разные трансформации (log1p, Box-Cox, diff) дают разный эффект на качество глобальной ML-модели.

Проект исследует, какая предобработка ряда полезна для CatBoost в зависимости от статистических характеристик ряда — волатильности, тренда, асимметрии.

---

## Структура репозитория

```
Time-Series/
├── config.py                   # все гиперпараметры
├── run_experiment.py           # точка входа
├── requirements.txt
│
├── src/
│   ├── __init__.py
│   ├── data.py                 # загрузка M4, выборка, train/test split
│   ├── clustering.py           # KMeans по статпризнакам, авто-подбор k по силуэту
│   ├── transforms.py           # none / log1p / boxcox / diff + обратные функции
│   ├── features.py             # лаги, rolling-агрегации, тренд, Фурье-сезонность
│   ├── models.py               # бейзлайны, AutoTheta, AutoETS, CatBoost
│   ├── metrics.py              # MAE, RMSE, MAPE, sMAPE, MASE
│   └── pipeline.py             # полный пайплайн
│
└── results/
    ├── analysis_results.ipynb  # визуализации и таблицы для отчёта
    ├── summary.csv             # метрики по всем трансформациям и моделям
    ├── cluster_breakdown.csv   # метрики отдельно по каждому кластеру
    └── conclusion.csv          # итоговая таблица: кластер - лучшая трансформация
```
